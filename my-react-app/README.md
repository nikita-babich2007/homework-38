# React + Vite

Простий додаток на React + Vite.

Ініціалізація проекту:
Створіть новий React проект за допомогою Vite:
npm create vite@latest my-react-app -- --template react
Перейдіть до каталогу проекту:
cd my-react-app
Встановіть залежності:
npm install
Запустіть проект:
npm run dev
